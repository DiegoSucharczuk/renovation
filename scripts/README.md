# Demo Data Seeding Script

סקריפט להוספת נתוני דוגמה לפרויקט בשביל הדגמה.

## מה הסקריפט מוסיף?

- **4 חדרים**: מטבח, סלון, חדר שינה, אמבטיה (עם אייקונים וסטטוסים שונים)
- **10 משימות**: משימות מגוונות מחולקות בין החדרים (צביעה, נגרות, חשמל, אינסטלציה וכו')
- **4 ספקים**: צבע, נגר, חשמלאי, אינסטלטור (עם פרטים מלאים)
- **5 תשלומים**: מקדמות ותשלומים בסטטוסים שונים

## איך מריצים?

### דרך 1: עם npx (מומלץ)

```bash
cd renovation-app
npx ts-node scripts/seed-demo-data.ts YOUR_PROJECT_ID
```

### דרך 2: עם התקנה גלובלית

```bash
npm install -g ts-node
cd renovation-app
ts-node scripts/seed-demo-data.ts YOUR_PROJECT_ID
```

### איך מוצאים את ה-PROJECT_ID?

1. היכנס לפרויקט שלך דרך האתר
2. העתק את ה-ID מה-URL:
   ```
   http://localhost:3000/dashboard/ABC123XYZ
                                   ^^^^^^^^^
                                   זה ה-ID שלך
   ```

## הערות

- הסקריפט עובד עם Firebase ישירות ולא דרך Next.js
- וודא שיש לך את כל משתני הסביבה ב-`.env.local`
- הסקריפט לא בודק אם הנתונים כבר קיימים - הוא יוסיף נתונים חדשים בכל הרצה
- אפשר להריץ כמה פעמים אבל זה יכפיל את הנתונים

## פתרון בעיות

### שגיאה: "Cannot find module 'firebase'"

התקן את Firebase:
```bash
npm install firebase
```

### שגיאה: "Cannot find module 'typescript'"

התקן TypeScript:
```bash
npm install -D typescript @types/node
```

### שגיאה: "Permission denied"

וודא שיש לך הרשאות כתיבה לאותו פרויקט ב-Firebase.
